# رسالة لــ رينادة

موقع ويب صغير رومانسي مكتوب باللغة العربية (تصميم موبايل-أول). يحتوي على نافذة بداية مرحة، بطاقات قابلة للقلب، قلوب عائمة في الخلفية، وزر موسيقى.

ملفات المشروع:

- `index.html` — صفحة الموقع (RTL، عربي).
- `styles.css` — كل الأنماط (مطوّر ليتناسب مع الشاشات الصغيرة أولًا).
- `script.js` — التفاعلات: المودال، سلوك زر "لا" الطريف، قلب البطاقات، توليد القلوب، ومشغل الموسيقى.
-- `assets/romantic.mp3` — (اختياري) ضع هنا أغنية رومانسيّة أجنبية بصيغة MP3. لا يتضمن المشروع ملف صوتي افتراضي؛ لو عندك ملف اغنية حطّه باسم `romantic.mp3` داخل مجلد `assets`.

تشغيل محلي:

1. افتح المجلد `D:/renada` في المتصفح (Double-click `index.html`) أو افتح عبر سيرفر محلي بسيط.
2. عند فتح الصفحة ستظهر رسالة: "قبل ما نبدأ... هتكملي ولا لا؟ 💗" — اضغطي "أيوه هكمل" لعرض المحتوى.
3. إن ضغطتِ "لا" سيحول الزر إلى "هكمل 😌" ثم يُكمل تلقائيًا (تفاعل ظريف).

إضافة موسيقى:

1. ضع ملف mp3 رومانسي في `D:/renada/renada2.mp3` إذا أردت استخدام ملفك الخاص (أو غيّر اسم الملف داخل `index.html` إذا اخترت اسمًا آخر).
2. اضغطي زر الموسيقى أعلى اليسار لتشغيل/إيقاف.

ملاحظة: إذا لم تضعي ملف MP3، الموقع يحتوي على مقطع رومانسي مولَّد داخل المتصفح (Web Audio API) يعمل تلقائيًا كبديل مجاني لذلك — لذلك لا حاجة لتحمّيل ملف إذا أردت تجربة سريعة. لإضافة الأغنية التي طلبتيها، ضعي ملف `renada2.mp3` في جذر المشروع (`D:/renada/renada2.mp3`).

ملاحظات تصميمية وملحوظات:

- تم استخدام خطوط Google (Cairo). النصوص الآن مكتوبة بالـ "فرانكو" (Franco-Arabic) — أي الحروف اللاتينية والأرقام التي تمثّل أصوات عربية (مثال: 7 = ح، 3 = ع، 2 = ء/أ).
- التصميم مرن ويعمل بشكل جيد على الهواتف. يمكن تعديل الأنيميشن وسرعاتها في `styles.css`.
- إن أردت ملف صوتي مضمنًا أو رابطًا خارجيًا، عدّل خاصية `src` في الوسم `<audio id="bgMusic">` ضمن `index.html` أو ضع الملف في المجلد `assets/`.

استمتع، وإذا تريد تخصيص النصوص أو إضافة إرسال رسالة أو تأثيرات إضافية أخبرني وأطبّقها.
